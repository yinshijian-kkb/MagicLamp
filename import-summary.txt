Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => MagicLamp/src/main/AndroidManifest.xml
* assets/ => MagicLamp/src/main/assets
* libs/android-support-v4.jar => MagicLamp/libs/android-support-v4.jar
* res/ => MagicLamp/src/main/res/
* src/ => MagicLamp/src/main/java/
